Proyectos Destacados

Tres en Raya

Un proyecto en pareja donde aplicamos lógica pura de programación. Creamos un sistema que detecta las jugadas del usuario y responde de forma estratégica para evitar la derrota. Nos centramos en la gestión del tablero y en crear una 'IA' que analice las posiciones ganadoras, siendo mi primer contacto serio con algoritmos de decisión en Java.

- Tecnologías: Java.

- Objetivo: Aprender lógica de juego y organización del código.

Sistema de fichajes

Desarrollé en pareja un sistema completo para gestionar una liga deportiva. El programa permite crear equipos desde cero, fichar jugadores y llevar un control detallado de su estado (activos o lesionados). También incluimos la gestión de presidentes para que el sistema refleje la estructura real de un club. Fue un reto excelente para trabajar con la lógica de estados y la organización de datos en Java.

- Tecnologías: Java.

- Objetivo: Dominar la estructura de clases y objetos (POO) para resolver problemas.

Netflix

He recreado la página principal de Netflix cuidando cada detalle visual, desde la película destacada hasta el famoso 'Top 10' y las listas por categorías. El reto principal fue lograr una estructura idéntica a la plataforma original, utilizando Flexbox y Grid para que los carruseles de películas sean fluidos y totalmente responsivos.

- Tecnologías: HTML5, CSS (Flexbox/Grid), JavaScript.

- Objetivo: Crear un diseño moderno y mejorar mis habilidades de maquetación.
